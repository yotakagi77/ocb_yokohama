## 途中経過 (gnuplot関連)
1. 以下のサイトを参照した．
   - [Gnuplot公式](http://www.gnuplot.info/)
   - User Manualを冒頭だけ読んだ．
   - [gnuplotスクリプトの解説 by 米澤先生](https://sk.kuee.kyoto-u.ac.jp/person/yonezawa/contents/program/gnuplot/index.html)を読んだ．
2. 作りたいグラフ・スクリプトのリスト
   - ヒストリーデータの時間変化(残差や抵抗係数の収束など)
   - エネルギースペクトル(両対数グラフ)
   - 相関関係のグラフ(散布図)
   - 相関関係のグラフ(3x3個のグラフ)
   - エラーバー付きの折れ線グラフ
   - 特殊関数
      - ベッセル関数
      - tanhと計算格子の関係
   - アニメーショングラフ(コーシー・ポアソン波，浅水波の伝播)
   - 等高線図・コンター図(速度ポテンシャル・流れ関数)
   - ノイズ除去
   - 最小二乗法
   - 関数プロットから表データを書き出す
   - 箱ひげ図
3. メモ
   - gnuplotのテスト(test)
   ```
     set term png
     set output "test.png"
     test
     unset output
   ```
